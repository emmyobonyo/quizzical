## Built With

- JSX, CSS

# Set Up
## Clone This Repository
```
$ git clone https://github.com/emmyobonyo/quizzical.git
$ cd quizzical
$ git checkout feat-welcome-page
```

## Run Project
```
$ npm install
$ npm start
```

## Run APP in the Broswer
- Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
